125 - DNSSEC - Web

Now try an AXFR transfert:

TXT values seems to be a very long base64 string.

To get only the string and save the file I used: drill -t AXFR datastore.challenge-by.ovh @167.114.236.142 | grep "IN TXT" | egrep '[A-Z0-9]{24}.datastore.challenge-by.ovh' | awk '{ print $(NF+1), $NF; }' | tr -d ' "\n' | base64 -di > file.ext:

  • grep "IN TXT" keeps only lines with TXT type
  • egrep '[A-Z0-9]{24}.datastore.challenge-by.ovh' removes non data names
  • awk '{ print $(NF+1), $NF; }' display the last field
  • tr -d ' "\n' removes spaces, new lines, and double quotes
  • base64 -di decodes teh base64
  • > file.ext saves the file
