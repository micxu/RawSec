Description

[This][this] seems to be recieving some sort of transmission. Our experts have been working around the clock trying and figure out what the hell it means with no hope of getting to the bottom of it. You're our only hope. [this]:https://play.icec.tf/problem-static/rip_2067f9686b4d07eea2cac19b9c6588b2abac16500135901ce8781e4ccc262446

Solution

  1. Check file type:

  2. Execute it: http://asciinema.org/a/276ngyyh05z80luxnqbyy95mq

  3. That's defintly not what we're looking for but only a lapse in concentration.

  4. Check what is hidden in this binary:

  5. We can see there is a zip with encrypted data (protected by password). Firstly we can extract this zip from the binary:

  6. Secondly we can bruteforce the password with patator, I first tried with 500-worst-passwords dict and then I was successful with john dict:

  7. Now we have just to unzip output_Sat_Aug_20_23_01_53_2016/zip/00002585.zip with bunny as password.

  8. And display output_Sat_Aug_20_23_01_53_2016/zip/rip.jpg to get the flag: IceCTF{1_Lik3_7o_r1P_4nD_diP_411_7He_ziP5}
